我们想要创建一个函数，在连续调用时将数字加在一起。

		add(1)(2);
		// returns 3

我们还希望能够继续为我们的连锁店添加数字。

		add(1)(2)(3); // 6
		add(1)(2)(3)(4); // 10
		add(1)(2)(3)(4)(5); // 15

单个调用应该返回传入的数字。

		add(1); // 1

我们应该能够存储返回的值并重用它们。

		var addTwo = add(2);
		addTwo; // 2
		addTwo + 5; // 7
		addTwo(3); // 5
		addTwo(3)(5); // 10
